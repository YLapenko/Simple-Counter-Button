# Simple-Counter-Button
### Ru
Это легкое PWA-приложение — простой и удобный счетчик кликов. Оно позволяет мгновенно фиксировать нажатия, поддерживает сброс результата и устанавливается прямо на экран смартфона как обычное приложение.
### En
This light PWA is a simple click counter that tracks taps in real-time. It features a reset option and can be installed on your smartphone for quick, app-like access.
